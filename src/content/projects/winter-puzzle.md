# WinterPuzzle / WinterPuzzle

Рождественская головоломка с семью открываемыми темами по три пазла: собранные изображения превращаются в игрушки для украшения ёлки.

A Christmas puzzle game with seven unlockable themes and three puzzles per theme: completed images become ornaments used to decorate a tree.
